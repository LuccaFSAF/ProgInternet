1. R: Ao acessar o site ou uma página web o vai passar por protocolos, 
        começando pelo IP onde o passará pelas quatro camadas, após isso, 
        ele vai estabelecer a comunicação através do HTTP, no qual, fará a 
        comunicação pelo servidor.

2. R: A frase está incorreta, pois o HTML é uma linguagem para formatação 
        de texto para WEB e CSS é é uma linguagem de formatação gráfica para WEB.

3. a)R: A primeira opção é utilizando a tag link, pois com ela é possivel
        chamar a ferramenta, como por exemplo: link rel="blabla" href="meucomendo.css",
        assim irá fazer a conexão entre o arquivo.

        A segunda opção é fazendo direto pelo arquivo html, onde a cima do corpo do
        irá chamar uma tag style, como por exemplo:
        style{
            *{
                border: 1px solid black;
            }
        } 
        e colocará bordas em todas as tags do corpo.

        A terceira opção seria fazer direta pela tag inserindo junto a tag style.

        c)R: Os posicionamentos funcionam átraves de especificações como por exemplo: 
        .container{
            display: fixed;
            left: 50%
            bottom: 10%
        }
        no qual leva o container para 50% à esquerda e 10% para baixo da página WEB.
